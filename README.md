# Naukri Resume Update Script

This script automates the process of updating your resume on Naukri.com using Selenium.

## Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install selenium webdriver-manager

## Jenkins useful Commands

- Check version - `jenkins --version`

- Start - `brew services start jenkins-lts`

- Stop - `brew services stop jenkins-lts`

- Restart - `brew services restart jenkins-lts`

- Update - `brew upgrade jenkins-lts`

## Puppeteer `headless` Option — Possible Values

| Value | Behavior |
|-------|----------|
| `false` | Full visible Chrome window opens (useful for debugging) |
| `true` | Old headless mode (legacy, being deprecated) |
| `'new'` | New headless mode — introduced in Chrome 112+, uses the real Chrome engine but without a GUI |
